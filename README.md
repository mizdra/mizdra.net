# mizdra.net

ほーむぺーじです。

## How to development

### Requirement

- Node.js v16+

### Setup

```console
$ git clone git@github.com:mizdra/mizdra.net.git
$ cd mizdra.net
$ pnpm i
$ pnpm dev
```

- `pnpm dev`: Start development server
- `pnpm lint`: Lint codes
- `pnpm build`: Build for production
- `pnpm start`: Start production server
